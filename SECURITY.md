# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within dome-icile, please do NOT open an issue. Email the Coding Squad directly.

## API Key Safety

* **Client-Side Only:** This application runs client-side. Your API keys are stored in browser memory/local storage only if you choose to save them.
* **No Analytics:** We do not track your prompts or your mind palace data. Your thoughts remain in your dome.
